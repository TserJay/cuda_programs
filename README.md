# CUDA Programs

A small repository for learning and experimenting with CUDA kernels. It is still a work in progress.

Current modules:

- `reduce`: reduction experiments
- `gemm`: basic matrix multiplication
- `permute`: tensor permutation and indexing practice

## Structure

```text
cuda_programs/
├── README.md
├── reduce/
├── gemm/
└── permute/
```

## Notes

- The code is mainly for learning and experimentation
- Some operators may have multiple versions for comparison
- The current main focus is the `reduce/` directory

## Getting Started

If you want to start with reduction, begin with:

1. `reduce/reduce.cu`
2. `reduce/reduce.py`
3. `reduce/README.md`
